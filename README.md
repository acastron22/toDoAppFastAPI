# toDoAppFastAPI

\*\* Criando a virtual machine
python -m venv ToDoApp

\*\* iniciando a virtual machine
ToDoApp\Scripts\activate

\*\* instalar fastapi e uvicorn
pip install fastapi
pip install "uvicorn [standard]"

\*\* desativar a vm
deactivate

\*\* install SQLAlchemy
pip install sqlalchemy

\*\* rodar o projeto
uvicorn main:app --reload
